# Lottie Animation Replacement Guide

## How to Replace Lottie Animations

1. Visit https://lottiefiles.com/
2. Search for animations matching each title below
3. Click on an animation you like
4. Click "Copy URL" or copy the JSON URL
5. Replace the `src="..."` URL in the HTML file

## Example Replacement

**Before:**
```html

```

**After (with your chosen animation):**
```html
<lottie-player src="https://assets5.lottiefiles.com/packages/lf20_YOUR_NEW_ID.json" ...></lottie-player>
```

## Tips
- Make sure the animation is set to "Free" or you have access
- Test the animation URL in your browser first
- Keep the same size attributes (150px for features, 120px for achievements)
- All animations should have: `background="transparent" speed="1" loop autoplay`
