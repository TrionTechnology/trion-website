## Steps to Download and Save

1. Visit https://lottiefiles.com/
2. Search for an animation matching the title
3. Click on the animation you like
4. Click the "Download" button
5. Select "Lottie JSON" format
6. Save the file with the exact name listed above in the correct folder

## Example

For "Fast Development Timeline":
- Go to LottieFiles.com
- Search "rocket" or "speed"
- Download as JSON
- Save as: `animations/features/fast-development.json`
